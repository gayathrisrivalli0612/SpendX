SpendX is a modern and minimalistic personal finance web application designed to help users efficiently track, analyze, 
and manage their daily expenses. The application focuses on simplicity, elegant design, and structured financial insights
while maintaining a smooth and intuitive user experience.
SpendX enables users to record expenses, monitor monthly budgets, analyze spending patterns, and track savings progress.
The system integrates modern front-end web development concepts with structured programming and data handling to deliver a 
practical financial management solution.
